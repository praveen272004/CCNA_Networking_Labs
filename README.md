# Computer Networks Lab (Cisco Packet Tracer)

This repository contains Cisco Packet Tracer lab experiments completed as part of the **CCNA Laboratory**. Each folder contains the network topology design, configuration screenshots, and output screenshots demonstrating the successful implementation of various networking concepts.

## Repository Structure

```text
.
├── ACL_Security
├── DHCP_Router
├── DHCP_Server
├── IIS_DNS_Server
├── Inter-VLAN_Routing_ROAS
├── Inter-VLAN_Routing_SVI
├── OSPF_BR_BDR
├── Ping_Basic
├── RIP (V1 & V2)
├── Router_CLI
├── Static_Route
├── Subnetting
└── VLAN_Trunking
```

## Lab Experiments

| No. | Lab Name | Description |
|:---:|----------|-------------|
| 1 | ACL Security | Configure Standard and Extended Access Control Lists (ACLs) to secure network traffic. |
| 2 | DHCP Router | Configure a Cisco router to function as a DHCP server for automatic IP address allocation. |
| 3 | DHCP Server | Configure a dedicated DHCP server to dynamically assign IP addresses to clients. |
| 4 | IIS & DNS Server | Configure IIS (Web Server) and DNS services for name resolution and web hosting. |
| 5 | Inter-VLAN Routing (ROAS) | Configure Router-on-a-Stick (ROAS) for communication between multiple VLANs. |
| 6 | Inter-VLAN Routing (SVI) | Configure Switch Virtual Interfaces (SVIs) to enable inter-VLAN routing on a Layer 3 switch. |
| 7 | OSPF BR/BDR | Configure OSPF and observe the Designated Router (DR) and Backup Designated Router (BDR) election process. |
| 8 | Ping Basic | Verify basic network connectivity using ICMP Ping. |
| 9 | RIP (V1 & V2) | Configure Routing Information Protocol (RIP) Version 1 and Version 2 for dynamic routing. |
| 10 | Router CLI | Perform basic router configuration using Cisco IOS Command Line Interface (CLI). |
| 11 | Static Route | Configure static routing between different networks. |
| 12 | Subnetting | Design IPv4 subnetworks using subnetting techniques. |
| 13 | VLAN Trunking | Configure VLANs and trunk links to allow VLAN traffic between switches. |

## How to Use

1. Clone this repository.
2. Open the desired lab folder.
3. View the topology diagram and configuration screenshots.
4. Open the Packet Tracer (`.pkt`) file using Cisco Packet Tracer.
5. Verify the configuration using the provided output screenshots.

```bash
git clone https://github.com/praveen272004/CCNA_Networking_Labs.git
cd CCNA_Networking_Labs
```
